# 程序依赖项

## 识别模型共享库依赖项

* CUDA Toolkit
  * cublas64_13.dll 或 cublas64_12.dll
  * cublasLt64_13.dll 或 cublasLt64_12.dll
  * cufft64_12.dll
* CUDNN
  * cudnn_engines_precompiled64_9.dll
  * cudnn_engines_runtime_compiled64_9.dll
  * cudnn_graph64_9.dll
  * cudnn_heuristic64_9.dll
  * cudnn_ops64_9.dll
  * cudnn64_9.dll
